# Proyecto Base de Datos - Maniquíes

## Orden de ejecución
1. Ejecutar `create.sql` para crear todas las tablas con sus tipos de datos.
2. Ejecutar `alter_tables.sql` si hay modificaciones adicionales.
3. Ejecutar `inserts.sql` para cargar los datos de prueba (piezas y maniquíes).

## Notas
- Todas las tablas tienen PRIMARY KEY.
- Se incluyen FOREIGN KEY para asegurar integridad referencial.
- Los campos obligatorios están definidos con `NOT NULL`.
